## NEXTJS Meetups app

In this app the user could see all events loaded by other users too and create new ones.
The database used to store datas is MongoDb.
This app is build using old nextjs pages router.

The package used are:
- "mongodb": "^6.3.0",
- "next": "14.1.0",
- "react": "^18",
- "react-dom": "^18"
